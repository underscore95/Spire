# Building

Spire is a CMake project using C++23.

The project depends on assimp, glfw, glm, glslang, imgui, spdlog, stb, vulkan, vulkan-memory-allocator.

## Steps to build
1. Run build.py in Spire/Libs, this builds all the libraries in debug and release mode and may take about 6 minutes to complete.
2. Reload CMake project
3. Build target Spire, this may take 1-2 minutes.
